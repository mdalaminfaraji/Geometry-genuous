1) What is DOM and what is the purpose of DOM?

Answer: Document object Model is full meaning of dom. We use dom then we can access and change all the elements of an HTML document. When a web page is loaded, then the browser creates a Document Object Model of the page.

The purpose of dom is given bellow:

1) we use dom then we can change all the HTML elements in the page

2) we can change all the HTML attributes in the page

3) we can change all the CSS styles in the page

4) we can remove existing HTML elements and attributes

5) we can add new HTML elements and attributes

6)we can react to all existing HTML events in the page

7) we can create new HTML events in the page

8) The HTML DOM is a standard for how to get, change, add, or delete HTML elements.

3) What is event bubble?

Answer: Event bubbling is a type of DOM event. when an event occurs in an element inside another element, and both elements have registered a handle for that event.when an element receives an event, and that event bubbles up to its parent elements in the DOM tree until it gets to the root element.

4)What is a callback function and why will you use it?

Answer: A callback function passed parameter to another function.This technique allows a function to call another function. A callback function can run after another function has finished.

The benefit of using a callback function is that you can wait for the result of a previous function call and then execute another function call.
